SQL Inner JOIN

The inner JOIN keyword in SQL is a specific way of merging tables in an SQL database. The syntax for calling the inner JOIN method is below:

SELECT column_name(s)
FROM table1
INNER JOIN table2
ON table1.column_name = table2.column_name;

Walking through each line of the sample code above will allow us to understand inner JOIN better. The first line ~ SELECT column_name(s) ~ shows what we want the database to feed back to us after the rest of the code executes. The code then joins specifies the table we want to look at which is in this case ~ table1 ~. The third line is where the inner JOIN keyword comes into play. Here, the SQL command instructs the function to join every record that have matching values in both tables, as specified by the ~ ON ~ line of code. Overall, the sample command above searches through the database and returns the fields under column_name(s) from table1 and table2 where table1.column_name equals table2.column_name.
